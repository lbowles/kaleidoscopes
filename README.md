# kaleidoscopes
 
This is the code repository for Kaleidoscopes, a fully on-chain, procedurally generated, animated kaleidoscopes collection.

## Stack

### Backend

- Hardhat
- Typescript
- Typechain

### Frontend

- React
- wagmi
- RainbowKit

## Usage

### Cloning

```
git clone <repo name>
```

### `backend`

```
yarn
```

```
yarn start
```

### `frontend`

```
yarn
```

```
yarn start
```
